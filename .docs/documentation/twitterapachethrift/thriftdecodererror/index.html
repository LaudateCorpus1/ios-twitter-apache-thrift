Enumeration

ThriftDecoderError

Errors thrown when the thrift is decoded.

Declaration

enum ThriftDecoderError

Topics

Enumeration Cases

case codingKeyMissingIntValue(key: CodingKey)
CodingKey is missing the value
case keyOrderMismatch
The key of the field is not the next key in the thrift, possibly because of malformed thrift data or model discrepancy
case nonUTF8StringData(Data)
The string found while decoding is not UTF8 formatted, possibly because of malformed thrift or an encoding error.
case readBufferOverflow
The buffer overflowed while trying to read a field, possibly because of using the wrong model type
case undecodableType(type: Any)
The type is not encodable, example not conforming to ThriftDecodable
case unexpectedlyFoundNil
The decoder unexpectedly found nil while decoding
case uninitializedDecodingData
Internal buffer is uninitialized should only happen if using methods other then decode<T>(_ type: T.Type, from data: Data) for decoding
case unsupportedThriftType
The thrift data has a unsupported type value
